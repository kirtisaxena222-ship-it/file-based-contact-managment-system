## File-Based Contact Management System (C++)

A simple console-based Contact Management System using File Handling in C++.

## Features
- Add Contact
- View All Contacts
- Search Contact
- Data stored permanently in contacts.txt

## Concepts Used
- Classes & Objects
- File Handling (fstream)
- String Handling
- CRUD operations
- Menu Driven Program

## How to Run

Compile:
g++ main.cpp -o contact

Run:
./contact
